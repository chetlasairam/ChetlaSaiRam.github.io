Sudoku Solver using OpenCV

About

The "Sudoku Solver using OpenCV and Python" is a program that utilizes computer vision technology through OpenCV in Python to solve Sudoku puzzles in real-time. This program involves showing an unsolved Sudoku puzzle to a camera, which then captures the puzzle from a live video feed. The algorithm processes the puzzle, solves it, and displays the solved puzzle within the real-time video feed.

Used Technologies

  • OpenCV (Computer Vision Library)
  • Python Programming Language
  • Real-Time Video Processing
  • Sudoku Solving Algorithm
  • Image Recognition and Processing

Description

The Sudoku Solver using OpenCV and Python is a unique application that combines computer vision and algorithmic techniques. When presented with an unsolved Sudoku puzzle via a camera, the program captures the puzzle from the live video and processes the image using OpenCV. It identifies the puzzle, solves it using an algorithm, and then overlays the solution onto the real-time video, displaying the solved Sudoku within the video feed.
