Face Mask Detection using Deep Learning

About

The "Face Mask Detection using Deep Learning" system is an innovative technology designed to identify whether individuals are wearing face masks accurately. Employing deep learning algorithms, this system operates effectively even in low-light conditions. During the COVID-19 pandemic, this technology serves as a helpful tool to ensure adherence to safety measures by detecting face masks on individuals.

Used Technologies

  • Deep Learning Algorithms (Convolutional Neural Networks)
  • Image Processing - OpenCV
  • Low-Light Image Enhancement
  • Python, TensorFlow, Keras, Numpy

Description

The Face Mask Detection system using Deep Learning is a powerful tool that employs advanced technology to identify whether people are wearing face masks correctly. What makes it unique is its ability to work even when the lighting is not very good. This technology became especially important during the coronavirus outbreak as it helped authorities ensure that people were following safety guidelines by wearing masks properly, even in places where light conditions were not ideal.
